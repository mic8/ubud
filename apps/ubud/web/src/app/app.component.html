Introduction to the Angular Docs

Angular is an application design framework and develompment platform for creating efficient and sophisticated single-page apps.

These angular docs help you learn and use Angular Framework and develompment platform, from you first application to optimizing complex single-page apps for enterprises. Tutorials and guides include downloadable examples to accelerate yout projects
